Features

We offer a lot of features, that are not available anywhere else. Just take a look at the list below.

Markup

HTML5 is a software solution stack that outlines the properties and behaviors of web pages by implementing a markup based pattern to it. HTML 5 is the current version of the HTML standard, allowing us to build diverse and powerful applications.

Scripts

JavaScript, often abbreviated as JS, is a high-level programming language that conforms to the ECMAScript specification - often used in combination with HTML5 to run dynamic web applications supported by desktops, tablets and mobile phones.

Components

Bootstrap is a popular free and open-source front-end Web framework. It contains HTML and CSS-based design components for typography, buttons, forms, navigation, modals and more - in combination with optional JavaScript-based extensions.

Hosting

Front-end applications, like the one you see here, can be hosted on any popular hosting platform - including Amazon AWS, Google Cloud, DigitalOcean and others. You can buy a domain from a website like namecheap and point it to your server.

Browsers

Various web browsers are at use today. We always try to use Google Developer Tools to view our web site at various viewports, and run our apps with both Firefox, IE and Edge before publishing them online for everyone in the world to see.

Playground

We're not scared to experiment with new technologies, and neither should you. Learning new stuff - from Youtube, Udemy, Wikipedia or anywhere else - expands our horizons, and enables us to improve both as a web developer, and a team player.
